This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

This is a static website clone of 睿均工程 (Rui Jun Engineering), a high-pressure water jet drain cleaning service company based in Kaohsiung, Taiwan. The site is configured for GitHub Pages hosting.

## Structure

- `index.html` - Homepage
- `Profile.html` - About page
- `Service.html` - Services page
- `Category.html` - Experience/projects listing
- `news.html` - News page
- `movie.html` - Videos page
- `contact.html` - Contact page
- `product-N00X.html` - Individual project detail pages (N001-N008)
- `assets/` - CSS, JS, and fonts (Bootstrap, Font Awesome, custom styles)
- `ec99/` - Additional CSS and JS (jQuery, shopping tab)
- `images/` - All images including banners and product photos
- `js/` - Additional JavaScript files
- `ShareFile/` - Validation scripts
- `rwd1350/store/f1/` - Favicon

## Deployment

This is a static site ready for GitHub Pages. Enable GitHub Pages in repository settings, pointing to the main branch root.
